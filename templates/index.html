Works

  • Password Checker

    One of the projects I have worked on in the past is a Python script that utilizes an API to check passwords against a database. Specifically, I used the API provided by the website haveibeenpwned.com to check if a given password has been compromised in any data breaches. This project was aimed at helping users ensure the safety of their passwords by checking them against a database of known compromised passwords.

    To implement this project, I first had to familiarize myself with the API and the data it provided. I then wrote a Python script that takes a password as input and sends a request to the API with the password's SHA-1 hash. The API then responds with a list of data breaches in which the password has been compromised, if any. The script then parses the response and presents the information to the user in an easy-to-understand format.

    This project not only allowed me to improve my Python skills but also gave me an opportunity to work with an API and learn about data breaches and password safety. It also helped me to understand the importance of password safety and the potential risks of using weak or compromised passwords.

  • Scrapping Hacker News

    Another project I have worked on in the past is a web scraper that scrapes articles from the website Hacker News. The goal of the project was to extract articles that have received 100 or more points, as they are considered to be popular and relevant. I used the Python library Beautiful Soup to scrape the data from the website.

    To implement this project, I first had to analyze the website's HTML structure and identify the relevant data that I wanted to extract. I then used Beautiful Soup to navigate through the HTML and extract the necessary information, such as the title, link, and number of points for each article. I also added a filter to only extract articles that have received 100 or more points.

    The data was then stored in a structured format, like a CSV file or JSON file, for further analysis or use. This project not only allowed me to improve my web scraping skills but also gave me an opportunity to work with Beautiful Soup and learn about web scraping techniques.

    Additionally, it also helped me to understand the importance of having the data structured for an easy analysis. It also helped me to understand the importance of the data and the potential uses of the same.

  • PDF to Audio

    Another project I have worked on in the past is a PDF to audio converter using Python, GTTS, PyPDF2, and PySimpleGUI. The goal of the project was to create a tool that can convert a PDF document into an audio file, allowing users to listen to the content of the PDF instead of reading it.

    To implement this project, I first had to install and import the necessary libraries such as GTTS, PyPDF2, and PySimpleGUI. I then wrote a Python script that takes a PDF file as input and uses PyPDF2 to extract the text from the PDF. The script then uses GTTS to convert the text into speech and save it as an audio file. PySimpleGUI was used to create a simple graphical user interface for the tool, making it easy for users to select the PDF and save the audio file.

    This project allowed me to improve my skills in Python programming and work with different libraries such as GTTS, PyPDF2, and PySimpleGUI. It also helped me to understand the importance of accessibility and the potential uses of the same. Additionally, it also helped me to understand the importance of the text to speech conversion and the potential uses of the same.
